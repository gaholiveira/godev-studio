# Ícones e Favicon

## Estrutura configurada no `layout.tsx`

- **Favicon:** `app/favicon.ico` — já existe; substitua pelo ícone final se quiser.
- **Ícone SVG:** `public/icon.svg` — favicon em SVG (escalável). Edite para usar o logo desejado.
- **Apple Touch Icon:** `public/apple-touch-icon.png` — adicione uma imagem **180×180 px** para ícone em dispositivos Apple (home screen, Safari).

## Como adicionar o Apple Touch Icon

1. Gere ou exporte uma imagem 180×180 px (PNG recomendado).
2. Salve como `apple-touch-icon.png` na pasta `public/`.
3. O metadata já aponta para `/apple-touch-icon.png`.
